# 후잉 API 샘플

써드파티에서 후잉 API에 접근하기 위한 샘플코드입니다. 기본적인 인증절차와 섹션, 항목, 거래들에 대한 기초 API 요청을 포함합니다.

## 주요 기능

- **인증**: OAuth 2.0 / OAuth 1.0a
- **섹션 조회**: 사용자의 가계부 섹션 목록 조회
- **재무제표 조회**: 재무상태표(BS), 손익계산서(PL)
- **거래 입력**: 복식부기 기반 거래내역 등록

## 기술 스택

- Svelte 4
- Rollup

## Install

```
$ npm i
$ npm run dev
```
